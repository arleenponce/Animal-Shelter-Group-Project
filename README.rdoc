== ShelterFriends

Welcome to ShelterFriends, a group project made by 5 LEARN Academy students for a final class project. This is a work in progress, feel free to open issues and create PRs to improve this project.

The goal was to create an elegant interface that would increase the likelihood of adoption for pets everywhere.

Application live online at <a href="https://aqueous-taiga-4911.herokuapp.com/">https://aqueous-taiga-4911.herokuapp.com/</a>, maintained by Emma Castor.

Default admin login in seeds.rb. Consider populating seed file with animals as well.
